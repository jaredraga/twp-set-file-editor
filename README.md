built by claude sonnet 4.6

this lets me manage and edit multiple Metatrader set files at once.
I'm using it with TWP Robot EA v1.11, MT5
But it probably works with any version, so long as set files are for the same versions (has the same columns/parameters)

to install and run locally:
`npm install`
`npm run dev`

Or you can use it online here:
[https://twp-set-file-editor.pages.dev/](https://twp-set-file-editor.pages.dev/)
Hosted on Cloudflare pages, runs locally on your browser. No server backend
